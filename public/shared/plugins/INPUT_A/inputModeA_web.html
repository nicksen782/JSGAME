HOW DO GAMEPADS AND BUTTON MAPPINGS WORK?


When a gamepad is connected it will be indicated in one one of the "Gamepad #" tabs.
In the same tab, a connected Gamepad will have either a red, yellow, or green icon to indicate it's readiness.
Red is disconnected.
Yellow is connected but no gamepad button map was found.
Green is connected and a gamepad button map was found.

There are two types of mappings: "internal" and "user".
"internal" are built-in mappings and include a mapping for a standard gamepad and also some gamepads that I either have or have created button mappings for.
"user" mappings are mappings that are created by the user. These are saved and loaded from HTML5 Local Storage and can be downloaded and restored if desired.
NOTE: The "user" mappings are checked first which means that a "user" button mapping can override an "internal" button mapping.
